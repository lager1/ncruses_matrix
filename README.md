# ncruses_matrix
simple matrix like program in ncurses

# dependencies
ncurses development library
(in debian `libncurses5-dev`)

# download & compilation
```
git clone https://github.com/lager1/ncruses_matrix.git
cd ncruses_matrix
g++ main.cpp -lncurses -o ncurses_matrix
./a.out
```
